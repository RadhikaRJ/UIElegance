Badge

Badge is a CSS component implemented to add notifications, messages, or statuses in different shapes and sizes. Badge components can be easily integrated with ListView, Avatar, and other container controls.

Copy below link into head tag of your HTML file:

Badge on Avatar

.avatar-x-small .badge-small-size

.avatar-small .badge-normal-size

.avatar .badge-large-size
